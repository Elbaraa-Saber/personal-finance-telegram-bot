# Personal Finance Telegram Bot

A Telegram bot for tracking personal income, expenses, reports, recent transactions, settings, language, and currency preferences.

The goal of this project is to provide a simple and practical personal finance assistant that can be used directly from Telegram.

## Features

- Add income transactions
- Add expense transactions
- View daily reports
- View weekly, monthly, yearly, and full reports
- Show recent transactions
- Delete the last transaction
- Change language settings
- Change currency settings
- Simple menu-based Telegram interface

## Tech Stack

- Node.js
- TypeScript
- grammY Telegram Bot Framework
- MongoDB
- Mongoose

## Project Status

This project is currently under active development.
